## Run Commands

### 1️⃣ Run on Connected Device (Recommended)
```bash
cd d:\Html\Arrow\Arrow_flutter

```

### 2️⃣ Build Release APK
```bash
cd d:\Html\Arrow\Arrow_flutter

```

### 3️⃣ Build for Web
```bash
cd d:\Html\Arrow\Arrow_flutter
flutter build web --release
```
Output folder: `build/web/`

## 🎯 Game Controls

- **Tap a block** to make it fly away (if path is clear)
- **Wrong move** = Block shakes + Lose 1 heart
- **All blocks cleared** = Level complete! Auto-advance
- **0 hearts** = Game Over → Restart
